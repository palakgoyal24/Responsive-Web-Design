Doctor's Searchbook Version 1.0 18/01/2017

GENERAL USAGE NOTES
---------------------

- This application allows the users to search for a Doctor in a
  particular City.

- Doctor's Searchbook is supported on all the web browsers.
  (Tested on Chrome, Microsoft Edge, Internet Explorer and Opera)

- If an 'Allow blocked content' notification occurs, users should 
  select Allow to procced with the application.
--------------------------------------------------------------------

System Requirememts
--------------------
All you need to do is to extract the contents from 'Doctor's Searchbook.zip' 
and run 'index.html' using any web-browser.
